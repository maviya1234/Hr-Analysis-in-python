# HR Analytics Dashboard

A comprehensive **HR Analytics Project** designed to analyze employee data, improve workforce management, and support data-driven HR decisions. This project uses **Data Analytics**, **Machine Learning**, and **Visualization** techniques to uncover employee trends, performance insights, attrition risks, and workforce statistics.

---

# 📌 Features

- 📊 Employee performance analysis
- 👨‍💼 Workforce distribution insights
- 📉 Employee attrition prediction
- 💰 Salary and compensation analysis
- 📅 Attendance and leave tracking
- 📈 Interactive dashboards and visualizations
- 🤖 Machine Learning models for prediction
- 🧹 Data cleaning and preprocessing

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/maviya1234/Hr-Analysis-in-python.git
cd hr-analytics
```

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

## 3️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ How to Run

## Run Analysis

```bash
python src/analysis.py
```

## Train Machine Learning Model

```bash
python src/model_training.py
```

## Run Application

```bash
python app.py
```
